# Elan Ruskin

## Bio

Elan Ruskin is a game developer and AI engineer who designed a scalable rule-matching dialog engine for Left 4 Dead, powered by a unified context dictionary of all game state. His system supports character memory, running gags, and additive design — new rules never require removing old ones. He authored Chapter 25 "Dialog."

## Key Contributions

- Designed the rule-matching dialog engine for Left 4 Dead
- Built a unified context dictionary of all game state powering dialog selection
- Engineered support for character memory, running gags, and additive rule design
- Authored Chapter 25 "Dialog"
