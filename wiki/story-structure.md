# Story Structure

## Definition
Story structure refers to the organizational framework that shapes a narrative — the sequence of events, the pattern of rising and falling action, and the relationship between story elements that makes a narrative recognizable and satisfying to an audience. Various models describe this structure, from the three-act paradigm to circular models like the [[harmon-story-circle]].

## Key Properties
- Structure is distinct from content: a story can have perfect structure but boring content, or exciting content but no structure
- Most structure models share a common pattern: equilibrium → disruption → journey → return → new equilibrium
- The [[harmon-story-circle]] emphasizes circular descent-and-return geometry, grounded in biological and psychological rhythms
- [[joseph-campbell]]'s [[monomyth]] describes a hero's journey through 17 stages that map to the 8 steps of Harmon's circle

## Examples
- Three-act structure: Setup, Confrontation, Resolution
- Harmon's 8-step circle: You → Need → Go → Search → Find → Take → Return → Change
- Campbell's Hero's Journey: Ordinary World → Call to Adventure → Threshold Crossing → Trials → Approach → Crisis → Treasure → Return → Resurrection → Return with Elixir
