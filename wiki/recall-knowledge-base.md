# Recall

## Definition

A SaaS knowledge base for personal knowledge management with 500,000+ users. Recall lets users save content from multiple sources (articles, YouTube, podcasts, PDFs, TikToks, Wikipedia, notes), get instant AI summaries, chat with their knowledge using models like GPT, Claude, or Gemini, and retain information through spaced repetition quizzes.

## Key Properties

- **Multi-source capture**: browser extension, web app, mobile app
- **AI summaries**: instant text and audio summaries with voice cloning option
- **Chat with knowledge**: pick context, pick AI model, switch mid-conversation
- **Auto-tagging**: smart tags that improve over time
- **Augmented browsing**: automatically links related ideas across saved content
- **Spaced repetition**: automated quiz generation on a personalized schedule
- **Data ownership**: full Markdown export, privacy-focused
- **MCP and API access**: available for integration with other tools
