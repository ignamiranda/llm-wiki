# Chris Rorden

## Bio

Chris Rorden is a neuroimaging researcher and prolific open-source developer. He created numerous tools widely used in the neuroimaging community including MRIcroGL (3D visualization), dcm2niix (DICOM to NIfTI conversion), nii2mesh (NIfTI to mesh conversion), and niimath (image processing). His tools are foundational in MRI/CT analysis workflows.

## Key Contributions

- Created nii2mesh — NIfTI volume to mesh conversion using Marching Cubes
- Created MRIcroGL and Surfice — 3D neuroimaging visualization tools
- Created dcm2niix — the de-facto standard DICOM to NIfTI converter
- Creator of niimath — NIfTI image processing
