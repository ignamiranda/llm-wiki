# Ink — Narrative Scripting Language

## Definition

Ink is a narrative scripting language created by Jon Ingold and Inkle for writing interactive fiction. It compiles to JavaScript (ink.js) and is used by Inkle's games including 80 Days, Sorcery!, and Heaven's Vault. Ink's core innovation is the Weave Structure — a system of knots, stitches, and diverts that enables non-linear narrative flow while maintaining authorial control over story progression.

## Key Properties

- Weave Structure: knots (scenes), stitches (sub-scenes), diverts (transitions)
- Glue and gather mechanics for flow control
- Choices with conditions, side effects, and tunnels
- Variable-based story state tracking
- Compiles to JavaScript for web and Unity integration
