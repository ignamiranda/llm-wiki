# Tristam MacDonald

## Bio

Tristam MacDonald (known as swiftcoder) is a game developer and Rust programmer specializing in graphics algorithms. He created the isosurface Rust library, which provides implementations of Marching Cubes, Dual Contouring, and their variants, along with deferred rasterisation from point clouds.

## Key Contributions

- Created the isosurface Rust library (marching cubes + dual contouring)
- Implemented GPU-side deferred rasterisation from point clouds
- Published educational documentation linking each algorithm to academic papers
