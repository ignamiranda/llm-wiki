# Kairos

## Definition

An unreleased memory system from Anthropic found referenced in leaked Claude Code source code. Described as an "always-on daemon that watches your project continuously, decides what's worth remembering, and consolidates old notes while you sleep." Not available in any public build as of mid-2026.

## Key Properties

- Always-on: runs continuously rather than being triggered by sessions
- Watches project activity to determine significance
- Autonomous decision-making about what to retain vs. forget
- Consolidation runs as a background process
- Indicates Anthropic is actively working on native solutions to [[context-rot]]
