# Xinyue Wei

## Bio

Xinyue Wei (Sarah Wei) is a researcher in computer vision and differentiable graphics. She created the DISO (Differentiable Iso-Surface Extraction) library, which provides high-performance CUDA implementations of differentiable Marching Cubes and Dual Marching Cubes for gradient-based mesh optimization.

## Key Contributions

- Created the DISO library — fast differentiable isosurface extraction in CUDA
- Author of Neumanifold (WACV 2025) — neural watertight manifold reconstruction
- Enabled gradient backpropagation through marching cubes for deep learning
