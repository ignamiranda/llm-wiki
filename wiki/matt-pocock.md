# Matt Pocock

## Bio / 简介

Matt Pocock is a prominent TypeScript educator and software engineer known for creating Total TypeScript, a comprehensive TypeScript learning platform. He is the author of the `mattpocock/skills` repository — an open-source collection of composable agent skills for AI coding agents that packages software engineering best practices into reusable workflows. Prior to his work on TypeScript education and AI-assisted development, he worked as a Developer Advocate at Vercel and has built widely-used open-source tools.

## Key Contributions / 主要贡献

- **Total TypeScript** — A comprehensive TypeScript learning platform with courses, exercises, and reference materials
- **Skills Repository** — Created and maintains the widely-adopted `mattpocock/skills` repository (~125k GitHub stars), defining the pattern of composable declarative skills for AI coding agents
- **TypeScript Open Source** — Contributor to TypeScript ecosystem tools, type utilities, and educational content
