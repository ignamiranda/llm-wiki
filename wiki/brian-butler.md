# Brian Butler

## Bio

Brian Butler is the co-founder of Freehold Games alongside [[jason-grinblat]] and co-creator of [[caves-of-qud]], a deeply procedural science-fantasy roguelike. He pioneered the use of Wave Function Collapse in texture mode for tile-based map generation, making Caves of Qud the first commercial game to use WFC. He has presented at GDC on the technical architecture of Caves of Qud's procedural generation systems, including the composition of multiple procgen techniques into a unified pipeline.

## Key Contributions

- First commercial use of Wave Function Collapse for game map generation
- Pioneered WFC texture mode (as opposed to tile mode) for tile-based maps
- Co-created Caves of Qud's chunky procgen architecture
