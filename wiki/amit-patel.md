# Amit Patel

## Bio / 简介

Amit J. Patel is the creator of Red Blob Games, a website with interactive tutorials on game development algorithms including procedural map generation, pathfinding, hex grids, and noise-based terrain. His tutorials are widely referenced in the game development community for their clear explanations and interactive visualizations.

## Key Contributions / 主要贡献

- Author of "Making maps with noise functions" — a definitive tutorial on noise-based terrain generation
- Author of "Polygonal Map Generation for Games" (2010) — the seminal guide on Voronoi-based map generation
- Created mapgen2, an open-source implementation of Voronoi map generation (ActionScript, later ported to JavaScript)
- Developed the initial terrain generator for Realm of the Mad God
- Wrote comprehensive guides on A* pathfinding and hex grid geometry
