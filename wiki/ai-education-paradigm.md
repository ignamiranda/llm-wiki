# AI Education Paradigm

## Definition

The shift in educational content creation driven by AI: instead of writing documentation and tutorials directly for humans, content creators write resources that teach AI agents, which then serve as personalized tutors for humans. An educator's value is in the "few bits" that AI agents cannot discover on their own — the hard-won insights, curriculum progression, and design decisions. Everything else, agents can explain better and with infinite patience.

## Key Properties

- Content written for agents first, humans second
- Markdown documents for agents replace HTML documentation for humans
- Agent skills encode curriculum progression and teaching hints
- Educators explain to agents; agents explain to humans
- The irreducible value is the 200 lines that represent years of insight
- Agents cannot yet discover the minimal essence — only understand it once shown

## Examples

Micro GPT: Karpathy created the 200-line implementation (the irreducible essence), but expects agents to handle all the educational delivery — explaining it to learners at their level, answering follow-up questions, and adapting to different backgrounds.
