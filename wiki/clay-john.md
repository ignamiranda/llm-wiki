# Clay John

## Bio

Clay John is the creator of the original *Eroded Terrain Noise* Shadertoy posted in 2018, which pioneered the non-simulated erosion filter technique. His work demonstrated that a noise function could produce eroded terrain with branching structure in a single-pass pixel shader, avoiding the need for slow water-drop simulation.

## Key Contributions

- Original *Eroded Terrain Noise* Shadertoy (2018) — the foundational implementation of the erosion filter technique
- Established the core approach: gradient-aligned stripes with cell-based pivot points and blending
