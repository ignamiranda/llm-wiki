## Source Summary / 来源摘要

`mattpocock/sandcastle` is a TypeScript library (`@ai-hero/sandcastle`) for orchestrating AI coding agents in isolated sandboxes. It provides a provider-agnostic system with built-in support for Docker, Podman, and Vercel sandbox providers, configurable branch strategies (head, merge-to-head, branch), and supports multiple agent providers (Claude Code, Pi, Codex, Cursor, OpenCode, Copilot). The library features a flexible prompt system with `{{KEY}}` substitution and `` !`command` `` dynamic context expansion, structured output extraction, and early termination via completion signals.
