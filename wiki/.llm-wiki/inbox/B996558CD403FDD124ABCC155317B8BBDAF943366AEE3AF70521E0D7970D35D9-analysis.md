## Source Summary / 来源摘要

Amit Patel's comprehensive guide (2010, updated 2025) on procedurally generating game maps using Voronoi diagrams instead of traditional tile grids. The article covers the full pipeline: generating Voronoi polygons with Lloyd relaxation, representing the map as dual graphs (Delaunay + Voronoi), defining coastlines and islands, computing elevation as distance from coast, generating rivers via downhill flow, calculating moisture from distance to fresh water, assigning biomes via the Whittaker diagram, and rendering with noisy edges.
