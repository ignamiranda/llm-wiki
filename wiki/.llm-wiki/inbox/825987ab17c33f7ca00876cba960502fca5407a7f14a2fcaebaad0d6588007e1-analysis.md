## Source Summary / 来源摘要

Shokunin (职人) is an open-source ecosystem by Elias Oulkadi providing 62 AI agent skills across 10 engineering domains, plus ChromaDB-based persistent memory and MCP servers for OpenCode, Claude Code, Cursor, Windsurf, Cline, and Continue.dev. It features a one-command installer, declarative self-updates with drift detection, and multi-strategy recall (vector + BM25 + temporal + RRF) with freshness decay.
