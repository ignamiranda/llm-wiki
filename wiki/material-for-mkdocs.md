# Material for MkDocs

## Definition / 定义

Material for MkDocs is a theme for the MkDocs static site generator that implements Google's Material Design principles. It is the most widely used MkDocs theme and provides extensive plugin APIs, social card support, search, navigation, and theming.

## Key Properties / 关键特性

- Built on MkDocs, written in Python
- Supports plugin system for extending functionality
- Version 9.0.0+ required by plugins like [[mkdocs-network-graph-plugin]]
- Full CSS variable theming with dark/light mode support
