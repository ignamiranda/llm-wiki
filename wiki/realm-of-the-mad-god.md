# Realm of the Mad God

## Definition

Realm of the Mad God (RotMG) is a bullet-hell massively multiplayer online game originally developed by Wild Shadow Studios. It commissioned Amit Patel's Mapgen2 system to procedurally generate its island-style maps using Voronoi polygon terrain with coast-distance elevation and Whittaker biome classification.

## Key Properties

- Maps are generated procedurally using Voronoi diagrams
- Uses elevation-from-coast-distance for terrain height
- Biomes assigned via Whittaker diagram (elevation × moisture)
- Mapgen2 was specifically built for RotMG's map generation needs
- Originally used Mapgen2 in Flash; later ported to HTML5
