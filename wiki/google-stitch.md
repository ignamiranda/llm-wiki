# Google Stitch

## Definition

Google Stitch is a platform and initiative by Google that introduced the DESIGN.md format — a plain-text design system document that AI agents read to generate consistent, brand-matching UI. The format is designed to be universally readable by any LLM without special parsing, serving as the design counterpart to `AGENTS.md` (which describes how to build a project).

## Key Properties

- **Format creator** — Google Stitch defined the DESIGN.md specification and format
- **AI-native design** — Designed specifically for AI agents to consume, unlike traditional design tokens or Figma exports
- **Open format** — Any project can adopt it by placing a `DESIGN.md` file in the project root
