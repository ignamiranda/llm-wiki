# Townscaper

## Definition

Townscaper is a procedural town-building game by Oskar Stålberg that uses Wave Function Collapse to generate colorful, organic island towns. Players place and remove voxel blocks, and WFC automatically fills in details — roofs, arches, stairs, pillars, overhangs — to create coherent buildings.

## Key Properties

- Uses WFC in real-time as the player edits
- Runs entirely in the browser (WebGL / Unity WebGL)
- WFC tilesets define how building elements connect (walls, roofs, windows, stairs)
- No goals or gameplay — pure creative sandbox
- Demonstrates WFC's potential for interactive, real-time content generation
