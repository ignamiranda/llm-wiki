# ProcFunc — Function-Oriented Abstractions for Procedural 3D Generation in Python

## Summary

Raistrick et al. (2026) introduce [[procfunclib]] (ProcFunc), a Python library that streamlines the creation, combination, and analysis of Blender-based procedural 3D generation code. Its function-oriented abstractions make procedural generation more accessible to both human developers and vision-language models (VLMs) for code editing. Includes a new indoor room generator built with the library.

## Content

ProcFunc addresses the challenge that procedural generation code for Blender is typically monolithic, hard to compose, and difficult for AI models to edit. It provides:

- **Function-oriented abstractions**: Decompose procedural generation into composable, testable function units
- **Blender integration**: Wraps Blender's Python API in higher-level functional primitives
- **VLM-friendly design**: The structured, compositional API allows VLMs to generate and edit procedural generation code reliably
- **Composition and analysis tools**: Utilities for combining generators and analyzing output properties
- **Reference generator**: A new indoor room generator built with ProcFunc, demonstrating its capabilities

The library is designed as a platform for both human developers writing procedural generators and VLMs generating 3D content through code editing.

## Key Takeaways

- Function-oriented API for Blender procedural generation
- Designed for both human and VLM use
- Demonstrates a new indoor room generator as a reference application
