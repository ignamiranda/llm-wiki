# VoltAgent Awesome Agent Skills — Curated Repository of 1000+ Agent Skills

## Summary

VoltAgent/awesome-agent-skills is the largest curated collection of agent skills for AI coding agents, featuring 1,000+ skills from over 50 official development teams and the community. Unlike bulk AI-generated repositories, this collection focuses on real-world skills created and used by actual engineering teams. It is compatible with Claude Code, Codex, Gemini CLI, Cursor, GitHub Copilot, OpenCode, Windsurf, and more.

## Content

The repository serves as a comprehensive directory of agent skills organized by source team, including:

- **Anthropic (Claude)** — 17 official skills for document creation (docx, pptx, xlsx, pdf), design (canvas, frontend, algorithmic art), branding, testing, and skill creation
- **VoltAgent** — 4 skills for the VoltAgent TypeScript framework (project setup, best practices, core reference, docs bundle)
- **Angular** — 2 skills for Angular development and new app creation
- **Google Gemini** — 4 skills for the Gemini API, Vertex AI, Live API, and Interactions API
- **Stripe** — 2 skills for Stripe integration best practices and SDK upgrades
- **Cloudflare** — 8 skills covering Workers, Durable Objects, the Agents SDK, email, web perf, and sandboxed execution
- **Trail of Bits (security)** — 21 skills covering smart contract security, static analysis, property-based testing, Semgrep rule creation, and audit workflows
- **Sentry** — 4 skills for error monitoring setup, issue fixing, and code review
- **Expo** — 11 skills for mobile app development, deployment, CI/CD, and native UI
- **Hugging Face** — 13 skills for ML workflows, datasets, model training, evaluation, and deployment
- **Netlify** — 12 skills for serverless functions, edge functions, blobs, databases, image CDN, and deployment
- **Google Workspace CLI** — 17 skills for managing Drive, Sheets, Gmail, Calendar, Docs, Slides, Tasks, and more
- **HashiCorp (Terraform)** — 11 skills for provider development, testing, module refactoring, and Terraform Stacks
- **And many more** — including Vercel (Next.js), Sanity, ClickHouse, Neon, Firebase, Flutter, OpenAI, Figma, Microsoft, DuckDB, MongoDB, Notion, Brave, Auth0, Apollo GraphQL, and others

The repository also features:
- **officialskills.sh platform** — A dedicated web platform for browsing and discovering official skills
- **Ecosystem integrations** — Promotion of the VoltAgent TypeScript framework for building AI agents with tool use, memory, and multi-agent coordination
- **Community contributions** — The most contributed Agent Skills repository, built and maintained together with the community

## Key Takeaways

- This is the largest curated listing of agent skills (1,000+), distinct from individual skill repositories
- Emphasizes real-world, team-built skills over AI-generated content
- Cross-platform compatible across all major AI coding agent runtimes
- Represents a maturing ecosystem where agent skills become a standard distribution mechanism
- The officialskills.sh platform provides a centralized discovery and browsing experience
