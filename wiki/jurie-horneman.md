# Jurie Horneman

## Bio

Jurie Horneman is a game AI designer and co-founder of ProcJam (Procedural Generation Jam), the community event that helped popularize procedural generation in independent game development. He developed the "story daemon" concept — an invisible system that monitors game state and injects narrative content adaptively — and created the Creepifier demo. He emphasizes that creative goals should drive technical choices in procedural storytelling. He authored Chapter 4 "Adapting Content to Player Choices" in Procedural Storytelling in Game Design.

## Key Contributions

- Co-founded ProcJam, the Procedural Generation Jam
- Developed the "story daemon" concept for adaptive narrative injection
- Created the Creepifier demo as a proof-of-concept for procedural content adaptation
