# VoltAgent Awesome DESIGN.md — Curated Repository of 73 Brand Design System Documents

## Summary

VoltAgent/awesome-design-md is a curated collection of 73 DESIGN.md files extracted from real brand websites. Each file follows the Google Stitch DESIGN.md format — a plain-text design system document that AI coding agents read to generate consistent, brand-matching UI. The collection spans 10 categories including AI platforms, developer tools, backend services, productivity SaaS, design tools, fintech, e-commerce, media, automotive, and retro web nostalgia.

## Content

The repository provides ready-to-use DESIGN.md files that define a brand's complete visual language:

- **Visual Theme & Atmosphere** — Mood, density, design philosophy
- **Color Palette & Roles** — Semantic name + hex + functional role
- **Typography Rules** — Font families, full hierarchy table
- **Component Stylings** — Buttons, cards, inputs, navigation with states
- **Layout Principles** — Spacing scale, grid, whitespace philosophy
- **Depth & Elevation** — Shadow system, surface hierarchy
- **Do's and Don'ts** — Design guardrails and anti-patterns
- **Responsive Behavior** — Breakpoints, touch targets, collapsing strategy
- **Agent Prompt Guide** — Quick color reference, ready-to-use prompts

Each site includes three files: `DESIGN.md` (the design system), `preview.html` (visual catalog), and `preview-dark.html` (dark surface catalog).
