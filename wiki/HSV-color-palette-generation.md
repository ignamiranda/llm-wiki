# HSV Color Palette Generation

## Definition

A technique for generating procedural color palettes by manipulating hue, saturation, and value independently. Complementary palettes use opposite hues (180° apart), triad palettes use equally spaced hues (120° apart), and analogous palettes use adjacent hues. Saturation and value ranges can be tuned independently. Used by Rune Skovbo Johansen in "A Study in Composition" for procedural landscape art that evokes different moods.

## Key Properties

- HSV color space for independent hue, saturation, and value control
- Complementary, triad, and analogous relationships
- Mood-driven palette selection
