# ProcJam — Procedural Generation Jam

## Definition

ProcJam (Procedural Generation Jam) is a community game jam focused on procedural content generation, co-founded by [[jurie-horneman]]. The event helped popularize procedural generation techniques in independent game development by creating a dedicated space for experimentation with generative systems, from text generation and level generation to audio and visual procedural techniques.

## Key Properties

- Community-driven game jam format
- Focus on generative/algorithmic content creation
- Helped popularize PCG in indie game development
- Co-founded by Jurie Horneman
