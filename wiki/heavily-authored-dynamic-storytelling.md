# Heavily Authored Dynamic Storytelling

## Definition

Richard Rouse III's middle path between pure procedural generation and fixed linear narrative. The system uses randomized starting states combined with hand-crafted content that changes enough for players to notice on subsequent playthroughs, without requiring a complex simulation.

## Key Properties

- **Discrete starting states** — Instead of a complex simulation, the game begins in one of several carefully designed starting configurations
- **Chapter titles as narrative anchors** — Each chapter's title provides context and framing for the procedural content within
- **Propaganda system** — PA lines are selected based on leader personality traits, creating faction-specific commentary
- **Desk/document propagation** — Documents and items are placed using weighted sets, ensuring variety while maintaining narrative coherence
- **Combinatorial ending system** — 3 narrative chunks combine to produce 20+ ending permutations, giving variety without exponential authoring cost

## Examples

Rouse describes the system used in *The Suffering* and similar games where each playthrough starts differently and characters react based on player reputation, but all content is hand-crafted.
