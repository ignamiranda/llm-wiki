# Gatis Kurzemnieks

## Bio

Gatis Kurzemnieks is a Technical Artist who created the 4-part tutorial series "Creating procedural game assets with Houdini" on RenderEverything.com. The series documents his exploration of procedural rock and cliff generation using Houdini, culminating in a fully automated game asset pipeline integrating Houdini PDG, Substance Designer (via Automation Toolkit), and RizomUV.

## Key Contributions

- **Procedural game asset pipeline** — documented a complete Houdini PDG-based pipeline from geometry generation through texturing and UV unwrapping to game engine export
- **Houdini + Substance integration** — demonstrated practical use of the Substance Automation Toolkit within Houdini PDG for headless texture generation
- **Toolchain cost analysis** — showed that the Houdini/Substance/RizomUV pipeline is accessible to individuals (~$639/year combined)
