# Caius Walker

## Bio / 简介

Caius Walker completed a BSc Honours degree in Computer Games Technology, where his final-year project designed and implemented a procedural sword generator in C#/Unity. The project demonstrated part-based procedural weapon assembly with CSV data storage, vertex-based mesh generation, and physics-integrated damage calculations. A second prototype is in development addressing limitations of the original.

## Key Contributions / 主要贡献

- First documented academic BSc Honours project on procedural sword generation
- Demonstrated physics-based damage calculation from procedurally generated geometry (kinetic energy, point of balance)
- Published two open-source GitHub repositories for the project
