# nii2mesh — NIfTI Volume to Mesh Converter

## Summary

nii2mesh by Chris Rorden is a lightweight C tool (no C++ dependencies) that converts NIfTI 3D voxel images to triangulated meshes using the enhanced Lewiner Marching Cubes algorithm, with options for pre-smoothing, iso-surface thresholding, mesh simplification, and multiple output formats.

## Content

Supports both the enhanced Lewiner Marching Cubes and the classic Cory Bloyd implementation. Features include multi-Otsu thresholding (dark/medium/bright), connected-component filtering, bubble filling, Laplacian post-smoothing with volume preservation, and quadric-error-metric mesh simplification. Outputs to GIfTI, MZ3, OBJ, PLY, FreeSurfer (.pial), STL, and VTK formats. Also supports atlas-based labeling for brain region meshes.

## Key Takeaways

- Pure C implementation (no C++ dependencies)
- Enhanced (Lewiner) or classic Marching Cubes
- Mesh simplification via quadric error metrics
- Supports atlas-based region meshing from label maps
- WebAssembly version available for browser use
