# Mapgen2 — Procedural Island Map Generator

## Summary
Mapgen2 is an interactive procedural island map generator by Amit Patel, built for the game Realm of the Mad God. It creates volcanic island-style maps using Voronoi polygons, with coast-distance elevation, center-island mountains, and simple biomes. The generator was originally a Flash application (2010) and later ported to HTML5/JavaScript (2017). A newer version, Mapgen4, supports painting terrain and simulating wind/rainfall.

## Key Design Choices
Mapgen2 was designed for specific game needs and makes no claim to realism:

- **Coastlines**: Designed to be interesting island shapes, not realistic continents
- **Mountains**: Placed in the center of islands, not in parallel ranges
- **Elevation**: Smooth, based on distance from coastline
- **Biomes**: Simple classification based on distance to coast and distance to water
- **Output**: Stylized map rendering reused from earlier experiments

These choices contrast with Mapgen4, which uses wind simulation, evaporation modeling, and rainfall to drive biome assignment.

## Technical Stack
- JavaScript with HTML5 Canvas
- Delaunator library (mapbox/delaunator) for fast Delaunay triangulation
- Apache2 license
- Icons drawn by hand, CC-BY licensed
