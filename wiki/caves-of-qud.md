# Caves of Qud

## Definition

Caves of Qud is an open-world science fantasy roguelike RPG developed by Freehold Games ([[jason-grinblat]] and [[brian-butler]]). Set in a far-future post-apocalyptic world where radiation has uplifted animals and plants to human intelligence, the game combines a handcrafted narrative with deeply simulative systems and extensive procedural generation. It is a tile-based game where every wall has a melting point, every creature has simulated biology, and the world's history is procedurally generated at the start of each playthrough.

## Key Properties

- **Hybrid narrative**: Handcrafted story weaved around simulative systems and procedural generation
- **Procedural history**: Each game generates a unique history centered on sultan dynasties, which physically manifests as explorable locations in the world
- **WFC map generation**: First commercial game to use Wave Function Collapse, in texture mode, for tile-based map generation
- **Simulative systems**: Physical simulation (dig anywhere, melting points), social systems (NPCs, factions, reputation), biological simulation
- **Chunky procgen architecture**: Composes multiple procedural techniques (noise, L-systems, Markov chains, WFC, replacement grammars) into a unified generation pipeline

## Examples

- Villages are generated by composing terrain noise, L-system vegetation, WFC buildings, Markov-chain names, and grammar-based history
- Sultan shrines in the world reference procedurally generated historical events that players can never fully verify
- Maps range from rectilinear palaces to organic cave networks, all generated from WFC samples
