# Christian Diller

## Bio

Christian Diller is a developer and researcher who created KinectFusionLib, a complete implementation of the KinectFusion real-time dense surface mapping system in modern C++14 and CUDA.

## Key Contributions

- Created KinectFusionLib — a well-documented, reusable KinectFusion implementation
- Created KinectFusionApp — a sample application demonstrating the library
- Published project website at christian-diller.de
