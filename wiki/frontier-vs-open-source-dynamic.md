# Frontier vs Open Source Dynamic

## Definition

The current dynamic in AI where closed frontier labs maintain a lead in model capability, but open-source models consistently follow with a narrowing gap (from ~18 months behind to ~6-8 months). This dynamic is considered healthy by Karpathy because it provides a common, democratized platform (like Linux for operating systems) while frontier labs continue pushing the edge of capability for problems that require it.

## Key Properties

- Frontier labs are ~6-8 months ahead in capability (gap narrowing)
- Open-source models are "good enough" for most consumer use cases
- Like Linux vs Windows/Mac: a shared open platform the industry trusts
- Capital expenditure advantage of frontier labs is the main moat
- Frontier intelligence will be needed for "Nobel Prize" level problems
- Open source will eat through the majority of basic use cases
- Centralization has poor track record — open-source provides power balance
