# KinectFusionApp — Sample Application for KinectFusionLib

## Summary

KinectFusionApp is a reference application for KinectFusionLib, providing camera implementations for recorded data and live depth sensors, with keyboard controls for mesh export and camera pose retrieval.

## Content

Implements camera data sources (recorded sequences and live OpenNI2 depth sensor). Provides keyboard shortcuts: 'p' for camera pose export, 'm' for mesh export, 'a' for all data export. Demonstrates the full KinectFusion pipeline from depth capture to surface mesh extraction via Marching Cubes.

## Key Takeaways

- Reference application for KinectFusionLib
- Supports recorded data and live OpenNI2 depth cameras
- Keyboard-controlled mesh and pose export
