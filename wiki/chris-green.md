# Chris Green

## Bio
Chris Green is a Valve engineer who authored the 2007 SIGGRAPH paper "Improved Alpha-Tested Magnification for Vector Textures and Special Effects." This paper introduced the use of signed distance fields for font rendering in real-time graphics, which became the primary reference for SDF font rendering in game engines.

## Key Contributions
- 2007 SIGGRAPH paper introducing SDF font rendering with simple GPU implementation
- Proposed multi-channel distance fields as a solution for sharp corners (without providing an implementation)
