# Nate (Open Brain)

## Bio

Nate is the creator of Open Brain, a structured memory system for AI that uses a SQL database as a durable knowledge store. He advocates for query-time knowledge synthesis (AI as reader) as opposed to compile-time synthesis (AI as writer, the Karpathy Wiki approach). His work focuses on building scalable, multi-agent-friendly memory architectures with auditable provenance, and he has proposed a hybrid model combining Open Brain's structured storage with Karpathy-style compiled wiki views.

## Key Contributions

- **Open Brain** — Structured SQL database memory system for AI agents
- **Hybrid Memory Architecture** — Proposed combining structured databases with compiled wiki views
- **AI as Reader Paradigm** — Query-time knowledge synthesis approach
- **Knowledge Contradiction Preservation** — Built tools for surfacing contradictions in structured data
