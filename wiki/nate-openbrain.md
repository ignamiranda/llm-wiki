# Nate B. Jones (Open Brain)

## Bio

Nate B. Jones is the creator of Open Brain (OB1), a structured AI memory system using Supabase (PostgreSQL + pgvector) as a durable knowledge store with an MCP server gateway. He advocates for query-time knowledge synthesis (AI as reader) as opposed to compile-time synthesis (AI as writer, the Karpathy Wiki approach). His work focuses on building scalable, multi-agent-friendly memory architectures with auditable provenance.

## Key Contributions

- **Open Brain (OB1)** — Supabase + pgvector + MCP server AI memory system (3.7k stars, 700 forks)
- **Extensions ecosystem** — 6 curated extensions (Household Knowledge → Job Hunt Pipeline) teaching progressive concepts
- **Community platform** — Discord community, Substack newsletter, companion Claude Skill / ChatGPT GPT / Gemini GEM
- **Recipes** — Import tools for ChatGPT, Perplexity, Obsidian, Twitter, Instagram, Google, Grok, Email
- **Skills** — Auto-Capture, Competitive Analysis, Research Synthesis, Meeting Synthesis, etc.
- **Hybrid Memory Architecture** — Proposed combining structured databases with compiled wiki views
