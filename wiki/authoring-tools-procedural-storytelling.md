# Authoring Tools for Procedural Storytelling

## Definition

Emily Short's framework for building authoring tools that enable writers — not just programmers — to create procedural narrative content. The approach centers on designing tools that respect writers' workflows while exposing the right amount of generative power.

## Key Properties

- **First five minutes principle** — A tool must produce a playable experience within the first five minutes of use; immediate gratification drives adoption
- **Sculptural / accretive writing** — Writers build narrative by adding, removing, and refining layers rather than writing linear text
- **Matching fiction to mechanics** — The authoring tool should encode the game's fictional domain so writers work in domain terms, not implementation details
- **Variety traps** — Adding more variants can *decrease* perceived quality if the new variants dilute the coherence of the content pool
- **Founder principle** — The first works built with a tool set precedent for what the tool is expected to do, sometimes for decades

## Examples

Short draws on her experience building tools for Versu and other interactive fiction platforms, where writers needed to define character knowledge, social rules, and narrative constraints without programming.
