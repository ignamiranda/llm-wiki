# Voyageur

## Definition

Voyageur is a game by Bruno Dias where the player travels between procedurally generated planets, each described through a filter stack architecture. The system combines a world model (what is true about the planet — atmosphere, life forms, resources) with grammatical templates, filtered through salience and style to produce unique, contextually appropriate descriptions.

## Key Properties

- Procedural planet generation with pre-seeded features
- Filter stack architecture balancing the "oatmeal problem" (uniqueness) and "brutalist building problem" (coherence)
- Ambient, low-agency exploration experience
- World model-driven text generation with corpus-based data
