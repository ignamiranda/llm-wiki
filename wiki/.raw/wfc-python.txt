An implementation of mxgmn/WaveFunctionCollapse in Python. WaveFunctionCollapse is an algorithm that generates bitmaps that are locally similar to the input bitmap. This is a translation into Python, based on the original implementation in C#. Minimal testing and optimization. An updated, re-implemented version of WaveFunctionCollapse in Python can be found at ikarth/wfc_2019f. Licensed under MIT. 141 stars, 25 forks. Python 100%.
