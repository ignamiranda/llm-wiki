WFC (Wave Function Collapse) and generic constraint satisfaction problem solver implementation for Godot 4. Features: Backtracking support, Multithreading, Learning from example. 2d WFC generator is able to infer rules from an example of a valid map. Supports different node types: TileMapLayer and TileMap (including some hexagonal tilemaps), GridMap. Supports tile probabilities. Contains a generic implementation of a solver capable of solving subclass of constraint satisfaction problems. Not yet implemented: 3d map generation, Wrapping, Lazy/dynamic generation, Global constraints. Licensed under MIT. 450 stars, 19 forks. GDScript 100%.
