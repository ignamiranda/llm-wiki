Single-file Wave Function Collapse library in C, plus a command-line tool. Supports the overlapping WFC method. The method takes a small input image and generates a larger output image which is locally similar to the input image. The WFC is often used for procedural map generation, but is not limited to this use-case. The library is very performant and includes a number of optimizations not found in other implementations. Generating a 128x128 image from cave.png (308 unique tiles) takes 371ms. Includes Rust bindings. Licensed under MIT. 367 stars, 22 forks. C 97.6%.
