# MkDocs Network Graph Plugin

## Definition / 定义

The MkDocs Network Graph Plugin is an open-source Python package by [[develmusa]] that extends [[material-for-mkdocs]] with an interactive force-directed network graph visualizing page relationships across the documentation site. It uses D3.js for client-side rendering and supports dual navigation modes.

## Key Properties / 关键特性

- Dual view modes: global site overview and per-page local connections
- CSS variable customization for full theme integration
- Zero client-side configuration — auto-generated from site structure
- Python 3.10+ with minimal build-time overhead

## Examples / 示例

- **Documentation portals**: Quickly visualize how documentation pages connect to help readers discover related content
- **Knowledge bases**: Render inter-page relationships as an explorable graph for large documentation sites
