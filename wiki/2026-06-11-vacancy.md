# Vacancy — Voxel Carving in C++

## Content

Vacancy implements Voxel Carving (Shape from Silhouette) in C++. Inputs are 2D binary silhouettes with camera parameters; output is a 3D mesh. Supports both naive voxel representation and KinectFusion-like TSDF fusion for smooth surfaces. Marching Cubes extracts the final mesh from the TSDF volume. Uses Eigen for math, with optional stb for image I/O and OpenMP for acceleration.

## Key Takeaways

- Voxel Carving + TSDF fusion
- Marching Cubes for mesh extraction
- Supports voxel and smooth mesh output
